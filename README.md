# Rock Paper Scissors Game 🪨📃✂️

A fun terminal-based Rock, Paper & Scissors game built with Python 
where you compete against the computer!

## Features
- Play against the computer with random choices
- Emoji icons for Rock 🪨, Paper 📃 and Scissors ✂️
- Displays win, lose or tie result after every round
- Option to continue or quit after each game

## How to Run
1. Make sure Python is installed on your computer
2. Run the script:
   python rock_paper_scissors.py
3. Enter your choice:
   - r = Rock 🪨
   - p = Paper 📃
   - s = Scissors ✂️
4. See if you beat the computer!
5. Choose y to play again or n to quit

## Technologies Used
- Python 3
- Random module
